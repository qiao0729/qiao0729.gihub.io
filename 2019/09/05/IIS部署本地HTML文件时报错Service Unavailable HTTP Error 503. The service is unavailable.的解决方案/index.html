IIS部署本地HTML文件时报错Service Unavailable HTTP Error 503. The service is unavailable.的解决方案

序言

又是一次踩坑之后的记录，上一篇博客记录了windows7中如何启动IIS服务，但是自己启动之后访问默认的网站 Default Web Site 都加载不出来。故开启了我的百度之路。终于找到了一篇博客。解决了自己的问题，特此记录，希望帮助大家少踩坑。

解决方法及步骤

刚开始访问默认的网站一直报错，停止之后又重新启动，依然还是报错。


找到应用程序池，点击进去，发现尽管我启动了 网站，但是在应用程序池中的状态是停止状态。

对于部署之后访问报错Service Unavailable HTTP Error 503. The service is unavailable的网站，在应用程序池中，鼠标右键选择高级设置，修改其标识为 NetworkService,如下图所示：

至此，再重新启动报错的网站，点击浏览，就可以正常访问了。非常感谢大佬提供的解决方案。
