使用node.js快速搭建本地服务器

前言

在学习Three.js的过程中，一般的功能可以通过编写代码之后直接运行html文件即可。但是在需要下载外部资源时，比如模型或者纹理图像，那么直接运行html代码是不行的，所以需要安装一个本地服务器。接下来我就来介绍一下如何用node.js搭建一个本地服务器。

一、安装node.js

先到node官网或者node中文网，下载并安装好node，安装好后可进入cmd输入node -v验证是否安装成功。如下图所示即为安装成功：

二、安装anywhere

安装了anywhere之后可以在任意目录启动一个静态服务器：

安装好node之后，打开cmd 输入npm install anywhere -g安装anywhere,成功之后如下图所示：

三、如何使用

搭建好之后如何使用呢：

  1. 在cmd页面，找到想要搭建服务器的路径，然后在该路径下输入 anywhere 8860,然后浏览器就会自动打开本地访问网址，一个简单的node服务器就搭建好了。

  2. 可以在vs code的终端输入anywhere 8860进行一个简单node服务器的搭建
